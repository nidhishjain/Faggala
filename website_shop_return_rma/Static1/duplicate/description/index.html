Website RMA - Return Orders Management

This module allow your customers to create return RMA order and management of RMA. Also allow then to view RMA.

Main Features:

- Customer can create RMA return order from Portal / My Account page.
- Customer can see list of RMA orders in Portal / My Account page.
- Sales team can manage RMA return orders in backend.
- Management of return pickings.
- Print RMA report - Print Return Report.
- For more details please watch Video.
